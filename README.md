# Machine-learning-assisted-gas-liquid-equilibrium-phase-diagram-of-binary-liquid-systems
Improvement of teaching experiments
By filtering, the data that meet the thermodynamic consistency were selected out. There were 60 sets of data. Adding the data of two pure components, there were a total of 62 sets of data. The machine learning model was trained and the boiling point temperature and liquid phase concentration were input to predict the gas phase concentration.
